## Quick Start

```bash
git clone https://github.com/hubshashwat/mimicpen.git
cd mimicpen/web-app
npm install
npm run dev
```

Open http://localhost:5173

## Get a Gemini API Key

1. Go to [Google AI Studio](https://aistudio.google.com/apikey)
2. Create a new API key
3. Paste it into the app
